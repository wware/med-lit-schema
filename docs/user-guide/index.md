# User Guide

This section is for people **using** the medical knowledge graph - querying data, integrating with applications, or building on top of the API.

## Contents

- [Quickstart](quickstart.md) - Get up and running in 5 minutes
- [Core Concepts](concepts.md) - Understand entities, relationships, evidence, and canonical IDs
- [Querying](querying.md) - REST API, GraphQL, and Python client
- [Storage Options](storage.md) - SQLite for development, PostgreSQL for production

## Who is this for?

- **Clinicians** asking questions like "What drugs treat BRCA-mutated breast cancer?"
- **Researchers** exploring "Which hypotheses have both supporting and refuting evidence?"
- **Application developers** building on top of the knowledge graph API
